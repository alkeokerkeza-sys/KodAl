Hyrje dhe Konfigurimi i C++

Kompilatorët, sintaksa, tipet, input/output

2 orë

C++ u krijua nga Bjarne Stroustrup në vitet 1980 si extension i gjuhës C. Sot fuqizon lojërat AAA (Unreal Engine), sistemet operative, softuerët financiarë dhe çdo aplikacion ku performanca është kritike.

Ndryshe nga Python ose JavaScript, C++ kompilohet drejtpërdrejt në instruksione makinerie. Kjo e bën shumë të shpejtë — por kërkon më shumë kujdes nga programuesi.

Setup: Instalo MinGW-w64 (Windows) ose GCC (Linux/Mac) dhe VS Code me shtesën C/C++.

// Programi i parë C++ #include <iostream> // libraria kryesore I/O #include <string> using namespace std; // nuk duhet të shkruajmë std:: gjithmonë int main() { // Outputi — cout dhe endl cout << "Pershendetje Bote!" << endl; // Variablat dhe tipet int mosha = 20; // numër i plotë double nota = 9.5; // numër dhjetor bool aktiv = true; // boolean string emri = "Arta"; // tekst char shkronja = 'A'; // një karakter cout << "Emri: " << emri << ", Mosha: " << mosha << endl; // Input nga përdoruesi string pyetje; cout << "Si te quajne? "; cin >> pyetje; cout << "Mire se erdhe, " << pyetje << "!" << endl; return 0; // 0 = programi u mbyll me sukses }
💡 Kompilimi: Ruaj si program.cpp, pastaj në terminal: g++ program.cpp -o program, pastaj ekzekuto me ./program. Çdo herë që ndryshon kodin duhet të rekompilohet!
2

Kontrolli i Rrjedhës dhe Funksionet

if/else, ciklet, funksionet, recursioni

2.5 orë

Kontrolli i rrjedhës në C++ është shumë i ngjashëm me gjuhë të tjera. Kujdes: C++ kërkon kllapkave {} për blloqet dhe ; pas çdo deklarate.

// if / else if / else int nota = 85; if (nota >= 90) { cout << "Shkelqyeshem!" << endl; } else if (nota >= 60) { cout << "Kaloi!" << endl; } else { cout << "Nuk kaloi." << endl; } // for loop for (int i = 0; i < 5; i++) { cout << "Hapi: " << i << endl; } // while loop int numri = 1; while (numri <= 5) { cout << numri * numri << " "; numri++; } // 1 4 9 16 25 // Funksion — duhet deklaruar PARA main() int shto(int a, int b) { return a + b; } // Funksion rekursiv — faktorial int faktorial(int n) { if (n <= 1) return 1; // rast bazë return n * faktorial(n - 1); // thirrje rekursive } // Overloading — funksione me emër të njëjtë, parametra të ndryshëm double shto(double a, double b) { return a + b; } // C++ zgjedh vetë cila të thirret bazuar te tipi!
💡 Deklarimi përpara: Nëse funksioni të tytësohet pas main(), duhet të vendosësh prototip para saj: int shto(int a, int b); — vetëm nënshkrimi, pa trupin.
3

Pointer-at dhe Menaxhimi i Memories

Stack vs Heap, pointerët, referencat, new/delete

3 orë

Pointer-at janë koncepti #1 i vështirë në C++ — por edhe #1 i fuqishëm. Një pointer është thjesht një variabël që ruan adresën e memories të variablit tjetër. Kuptoje dhe do kuptosh si funksionon kompjuteri.

// & = adresa e, * = vlera te adresa int x = 42; int* ptr = &x; // ptr mban adresën e x cout << x << endl; // 42 — vlera e x cout << &x << endl; // 0x... — adresa e x cout << ptr << endl; // 0x... — e njëjta adresë cout << *ptr << endl; // 42 — vlera TE adresa (dereferencing) *ptr = 100; // ndrysho x nëpërmjet pointer-it cout << x << endl; // 100! // Referencat — alias i variablit (C++ vetëm) int& ref = x; // ref IS x — jo kopje ref = 200; // x bëhet 200 // Alokimi dinamik — HEAP (ti kontrollon jetëgjatësinë) int* ptr2 = new int(55); // alokon memorie në heap cout << *ptr2 << endl; // 55 delete ptr2; // DUHET fshirë! Memory leak nëse jo ptr2 = nullptr; // shëno si null pas delete-s // Array me pointer int* arr = new int[5]; // array prej 5 int-ësh arr[0] = 10; arr[1] = 20; delete[] arr; // delete[] për array!
💡 Memory Leak: Nëse alokon me new dhe harron delete, memoria mbetet e zënë edhe pas mbylljes së bllokut. Programi rritet derisa ngrin. Në C++ modern prefero unique_ptr dhe shared_ptr — menaxhojnë delete automatikisht!
4

OOP me C++ — Klasat dhe Trashëgimia

Classes, constructors, inheritance, polymorphism

3.5 orë

C++ është gjuha OOP par excellence. Klasat bashkojnë të dhëna (atribute) dhe funksione (metoda) në një strukturë. public/private/protected kontrollojnë se kush mund të aksesojë çfarë.

class Karroca { private: // aksesueshem vetem brenda klasës string marka; int shpejtesia; double karburant; public: // aksesueshem nga jashtë // Constructor Karroca(string m, double k) { marka = m; karburant = k; shpejtesia = 0; } // Destructor — thirret kur objekti shkatërrohet ~Karroca() { cout << marka << " u fshi." << endl; } void shpejto(int vlera) { if (karburant > 0) shpejtesia += vlera; } void info() { cout << marka << ": " << shpejtesia << " km/h" << endl; } }; // Trashëgimia class MakineElektrike : public Karroca { private: int bateria; // % e baterisë public: MakineElektrike(string m, int b) : Karroca(m, 0) { bateria = b; } void karikо() { bateria = 100; } }; // Polymorphism me virtual class Forma { public: virtual double zona() = 0; // pure virtual — duhet override }; class Rreth : public Forma { double r; public: Rreth(double radius) : r(radius) {} double zona() override { return 3.14159 * r * r; } };
💡 virtual dhe override: Pa virtual, trashëgimia statike — C++ zgjedh funksionin bazuar te tipi i pointer-it, jo objektit. Me virtual, zgjedhja bëhet në runtime (dynamic dispatch) — kjo është polimorfizmi i vërtetë!
5

STL — Standard Template Library

vector, map, set, iteratorët dhe algoritmet

3 orë

STL është koleksioni i strukturave të të dhënave dhe algoritmeve të ndërtuara brenda C++. Kurrë mos ndërtosh vector-in ose sort-in nga zero — STL i ka të gjitha!

#include <vector> #include <map> #include <algorithm> #include <iostream> using namespace std; // vector — array dinamike vector<int> numrat = {5, 2, 8, 1, 9}; numrat.push_back(3); // shto në fund numrat.pop_back(); // fshi fundin cout << numrat.size(); // 5 // Iterim me range-based for (C++11) for (int n : numrat) cout << n << " "; // Algoritmet sort(numrat.begin(), numrat.end()); // rendit rritës reverse(numrat.begin(), numrat.end()); auto max_el = *max_element(numrat.begin(), numrat.end()); // map — çelës:vlerë (i renditur) map<string, int> notat; notat["Arta"] = 95; notat["Blerim"] = 78; notat["Çlirim"] = 88; for (auto& [emri, nota] : notat) { // structured binding C++17 cout << emri << ": " << nota << endl; } // unordered_map — çelës:vlerë (pa rend, por SHUMË i shpejtë) #include <unordered_map> unordered_map<string, int> freq; for (char c : "programim") freq[c]++;
💡 vector vs array: Gjithmonë prefero vector mbi array-t e zakonshme. Vector rritet dinamikisht, di madhësinë e vet, dhe punon me algoritmet e STL. Array-t janë të dobishëm vetëm kur madhësia dihet compile-time dhe performanca është kritike.
6

C++ Modern — C++17 dhe C++20

Smart pointers, lambda, auto, ranges

3 orë

C++ modern (C++11 dhe pas) ka ndryshuar rrënjësisht si shkruhet C++. Kodi i ri duhet të përdorë këto veçori — janë shumë më të sigurta dhe elegante.

#include <memory> // për smart pointers // Smart Pointers — auto-delete, pa memory leak! auto ptr = make_unique<int>(42); // delete automatik cout << *ptr << endl; // 42 // ptr fshihet automatikisht kur del nga scope — pa new/delete! auto ptr2 = make_shared<string>("Shperndarje"); // shared_ptr: shumë pronarë, fshihet kur të gjithë largohen // Lambda — funksione anonime auto dyfish = [](int x) { return x * 2; }; cout << dyfish(7) << endl; // 14 // Lambda me capture — kap variabla nga scope-i jashtë int baza = 10; auto shto_bazen = [baza](int x) { return x + baza; }; cout << shto_bazen(5) << endl; // 15 // Lambda me sort vector<string> emrat = {"Çlirim", "Arta", "Blerim"}; sort(emrat.begin(), emrat.end(), [](const string& a, const string& b) { return a < b; }); // auto — deduksion automatik i tipit auto x = 42; // int auto y = 3.14; // double auto z = "tekst"; // const char* // Optional — vlera mund të mos ekzistojë (C++17) #include <optional> optional<int> gjej(int x) { if (x > 0) return x * 2; return nullopt; // nuk ka vlerë }
💡 Rregulli i C++ Modern: Nëse shkruan new/delete drejtpërdrejt — ke problem. Prefero make_unique dhe make_shared. Kompilo gjithmonë me -std=c++17 ose -std=c++20.

// Skelet i Sistemit të Notave #include <iostream> #include <vector> #include <algorithm> #include <fstream> using namespace std; struct Studenti { string emri; vector<double> notat; double mesatare() { double shuma = 0; for (double n : notat) shuma += n; return shuma / notat.size(); } }; int main() { vector<Studenti> klasa; // Shto studentë klasa.push_back({"Arta", {9.5, 8.0, 9.0}}); klasa.push_back({"Blerim", {7.0, 6.5, 7.5}}); // Rendit sipas mesatares (zbritës) sort(klasa.begin(), klasa.end(), [](auto& a, auto& b) { return a.mesatare() > b.mesatare(); }); for (auto& s : klasa) cout << s.emri << ": " << s.mesatare() << endl; }
