1

Bazat e JavaScript-it

Variablat, tipet, operatorët dhe si JS funksionon në shfletues

2 orë

JavaScript është gjuha e vetme programimi natyrore e shfletuesit. HTML ndërton strukturën, CSS i jep pamje, JS i jep sjellje. Kur klikoni një buton dhe diçka ndodh — ajo është JavaScript.

Mënyra moderne e deklarimit të variablave është me let dhe const. Shmangni var — ka probleme scope-i.

// Variabla moderne const emri = "Arta"; // nuk ndryshon let mosha = 20; // mund të ndryshojë mosha = 21; // OK me let // Tipet kryesore const tekst = "Përshëndetje"; // string const numri = 42; // number const decimal = 3.14; // gjithashtu number const aktiv = true; // boolean const asnje = null; // null let papercaktuar; // undefined // Template literals (strings moderne) const mesazh = `Pershendetje, ${emri}! Ke ${mosha} vjec.`; console.log(mesazh); // Operatorë logjikë console.log(5 > 3); // true console.log(5 === "5"); // false — === krahason TIP gjithashtu console.log(5 == "5"); // true — == konverton tipin (SHMANGNI!)
💡 Gjithmonë përdor === (triple equals) në vend të == (double equals). Triple equals krahason si vlerën ashtu edhe tipin e të dhënave — shumë më i sigurt!
2

Funksionet dhe Scope-i

Deklaratat, shprehjet, arrow functions, closures

2.5 orë

JavaScript ka disa mënyra të shkruash funksione. Arrow functions janë sintaksa moderne dhe kompakte — i sheh kudo në kod profesional.

// Mënyra tradicionale function presto(a, b) { return a + b; } // Function expression const presto2 = function(a, b) { return a + b; }; // Arrow function — sintaksa moderne ⭐ const presto3 = (a, b) => a + b; const dyfish = x => x * 2; // një parametër, pa kllapa const pershendetje = () => "Hej!"; // pa parametra // Closures — funksion brenda funksioni function krijo_counter() { let count = 0; return function() { count++; return count; }; } const counter = krijo_counter(); console.log(counter()); // 1 console.log(counter()); // 2 console.log(counter()); // 3 — count "kujtohet"! // Scope: let/const janë "block scoped" if (true) { let x = 10; const y = 20; } // console.log(x); // ERROR — x nuk ekziston jashtë blokut
💡 Closures janë nga konceptet më të vështira por edhe të fuqishme të JS. Funksioni i brendshëm "kap" variablat e funksionit të jashtëm — edhe pasi ai funksion ka mbaruar ekzekutimin.
3

DOM Manipulation

Zgjedhja e elementëve, ndryshimi i përmbajtjes, trajtimi i ngjarjeve

3 orë

DOM (Document Object Model) është mënyra si JavaScript "sheh" HTML-in — si pemë objektesh. Çdo tag HTML bëhet objekt JavaScript që mund ta lexojmë, ndryshojmë ose fshijmë.

// Zgjedhja e elementëve const titulli = document.querySelector("h1"); const butonat = document.querySelectorAll(".buton"); const forma = document.getElementById("forma-kontaktit"); // Ndryshimi i përmbajtjes titulli.textContent = "Titulli i Ri!"; titulli.innerHTML = "Titulli me bold"; // Ndryshimi i stilit titulli.style.color = "#e07b54"; titulli.style.fontSize = "48px"; // Shtimi/heqja e klasave CSS titulli.classList.add("aktiv"); titulli.classList.remove("fshehur"); titulli.classList.toggle("highlight"); // shton nëse mungon, heq nëse ka // Ngjarjet (Events) — zemra e interaktivitetit const butoni = document.querySelector("#butoni-im"); butoni.addEventListener("click", function() { alert("Butonin u klikua!"); }); // Arrow function — mënyra moderne butoni.addEventListener("click", () => { butoni.textContent = "Klikuar!"; butoni.disabled = true; }); // Forma — parandalim i dërgimit default forma.addEventListener("submit", (event) => { event.preventDefault(); // mos rifresko faqen const emri = document.querySelector("#emri").value; console.log("Formulari u dërgua nga:", emri); });
💡 querySelector() është mjeti universal modern. getElementById() dhe getElementsByClassName() janë të vjetëruara — mëso querySelector() dhe querySelectorAll() dhe do t'i mbulojë të gjitha rastet.
4

Array-t dhe Objektet

Metodat, destrukturimi, spread/rest, JSON

2 orë

Array-t dhe objektet janë strukturat kryesore të të dhënave në JS. Metodat moderne si map, filter dhe reduce janë thelbësorë — i sheh kudo në kod React, Node.js etj.

const studentet = [ { emri: "Arta", nota: 9.5 }, { emri: "Blerim", nota: 7.0 }, { emri: "Çlirim", nota: 8.5 }, ]; // map — transformo çdo element const emenat = studentet.map(s => s.emri); // ["Arta", "Blerim", "Çlirim"] // filter — filtro bazuar te kusht const mire = studentet.filter(s => s.nota >= 8); // [{emri:"Arta",...}, {emri:"Çlirim",...}] // find — gjej elementin e parë që plotëson kushtin const arta = studentet.find(s => s.emri === "Arta"); // reduce — reduko array në një vlerë const shuma = studentet.reduce((total, s) => total + s.nota, 0); const mesatare = shuma / studentet.length; // 8.33 // Destrukturimi const { emri, nota } = studentet[0]; console.log(emri, nota); // Arta 9.5 const [i_pari, i_dyti] = emenat; console.log(i_pari); // Arta // Spread operator const studentet_re = [...studentet, { emri: "Diana", nota: 9.0 }]; // JSON const json_tekst = JSON.stringify(studentet); // objekt → string const objekti = JSON.parse(json_tekst); // string → objekt
💡 map, filter, find dhe reduce janë "funksionale" — kthejnë vlerë të re pa ndryshuar array-in origjinal. Kjo quhet immutability dhe është praktikë e mirë.
5

JavaScript Asinkron

Callbacks, Promises, async/await, fetch API

3 orë

JavaScript është single-threaded — ekzekuton një gjë në një kohë. Por shumë operacione (kërkesa rrjeti, leximi i skedarëve) marrin kohë. Kodi asinkron e lejon JS të vazhdojë punën ndërkohë që pret — pa ngrirë faqen.

// FETCH API — merr të dhëna nga interneti // Mënyra moderne me async/await ⭐ async function merr_kodin_e_ditës() { try { // fetch() dërgon kërkesë HTTP const pergjigja = await fetch("https://api.quotable.io/random"); // kontrollo nëse shkoi mirë if (!pergjigja.ok) { throw new Error("Kërkesa dështoi!"); } // kthe JSON-in const te_dhenat = await pergjigja.json(); console.log("Citati:", te_dhenat.content); console.log("Autori:", te_dhenat.author); // shfaq në faqe document.querySelector("#citati").textContent = te_dhenat.content; } catch (gabim) { console.error("Gabim:", gabim.message); document.querySelector("#gabim").textContent = "Nuk u arrit të lidhet."; } } // thërrite kur faqja ngarkohet merr_kodin_e_ditës(); // ose kur klikohet buton document.querySelector("#buton-kito").addEventListener("click", merr_kodin_e_ditës);
💡 async/await është shumë më i lexueshëm se Promises zinxhir me .then().catch(). Gjithmonë mbulo await me try/catch — kërkesat e rrjetit mund të dështojnë!

ES6+ — Veçoritë Moderne

Modulet, klasat, optional chaining, dhe më shumë

2.5 orë

ES6 (2015) dhe versionet pas tij sollën ndryshime revolucionare. Kodi modern JS është pothuajse i pakuptueshëm pa njohur këto veçori — do t'i shohësh çdo ditë.

// Modulet — organizo kodin në skedarë // skedari: math.js export const PI = 3.14159; export function shuma(a, b) { return a + b; } export default class Kalkulatori { /* ... */ } // skedari: main.js import Kalkulatori, { PI, shuma } from "./math.js"; // Klasat (ES6) class Karroca { #shpejtesia = 0; // fushë private (ES2022) constructor(marka) { this.marka = marka; } shpejto(vlera) { this.#shpejtesia += vlera; return this; // zinxhirim metodash } get info() { return `${this.marka}: ${this.#shpejtesia} km/h`; } } const BMW = new Karroca("BMW").shpejto(50).shpejto(30); console.log(BMW.info); // BMW: 80 km/h // Optional Chaining — shmang gabimet "cannot read property of null" const perdoruesi = { profil: { adresa: { qyteti: "Tiranë" } } }; const qyteti = perdoruesi?.profil?.adresa?.qyteti ?? Projekti Final — Aplikacion Interaktiv Ueb

Ndërto një aplikacion të plotë duke përdorur API-t reale

7 orë

Do të ndërtojmë një aplikacion moti (Weather App) real duke lidhur me API publike. Aplikacioni do të marrë vendndodhjen e përdoruesit, do të kërkojë të dhëna moti live dhe do t'i shfaqë me animacione.

// Weather App — Struktura kryesore async function merr_motin(qyteti) { const API_KEY = "çelësi_yt_nga_openweathermap"; const URL = `https://api.openweathermap.org/data/2.5/weather?q=${qyteti}&appid=${API_KEY}&units=metric&lang=al`; const resp = await fetch(URL); const data = await resp.json(); shfaq_motin({ qyteti: data.name, temperatura: Math.round(data.main.temp), pershkrimi: data.weather[0].description, lagështia: data.main.humidity, era: data.wind.speed, ikona: data.weather[0].icon }); } function shfaq_motin(moti) { document.querySelector("#qyteti").textContent = moti.qyteti; document.querySelector("#temperatura").textContent = `${moti.temperatura}°C`; document.querySelector("#pershkrim").textContent = moti.pershkrimi; document.querySelector("#ikona").src = `https://openweathermap.org/img/wn/${moti.ikona}@2x.png`; } // Kërko me Enter document.querySelector("#input-qytet") .addEventListener("keypress", e => { if (e.key === "Enter") merr_motin(e.target.value); });
