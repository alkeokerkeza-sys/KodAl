1

Bazat e CSS dhe Selektorët

Si i lidhim stilet me elementet HTML

2 orë

CSS (Cascading Style Sheets) është gjuha me të cilën "vishen" faqet HTML. Pa CSS, çdo faqe duket si dokument teksti i thjeshtë. CSS i tregon browserit: ky tekst duhet të jetë i kuq, kjo kuti duhet të ketë 20px hapësirë, ky buton duhet të ketë sfond blu.

Ka 3 mënyra si e lidhim CSS-in me HTML-in:

/* 1. Menyra me e mire - file i vecante */ <link rel="stylesheet" href="stili.css"> /* 2. Brenda <style> ne <head> */ <style> p { color: blue; } </style> /* 3. Inline - direkt ne element (shmangie!) */ <p style="color: blue;">Tekst blu</p>

Selektorët janë si "çelësat" — tregojnë cilit element i japim stilin:

/* Selektori i tagjit - te gjithe p */ p { color: #333; } /* Selektori i klasës - elementet me class="kartë" */ .karte { background: white; border-radius: 8px; } /* Selektori i ID - elementi me id="hero" */ #hero { height: 100vh; } /* Selektori i kombinuar */ .karte p { font-size: 14px; }
💡 Klas vs ID: Klasa (.karte) mund të përdoret tek shumë elementë. ID (#hero) duhet të jetë unik — vetëm një element në faqe. Prefero gjithmonë klasat!
2

Ngjyrat, Tipografia dhe Njësitë

Fontet, madhësitë dhe sistemi i ngjyrave

2 orë

Ngjyrat në CSS mund të shkruhen në disa mënyra. Hex është forma më e zakonshme — 6 karaktere që kodojnë të kuqen, të gjelbërtën dhe të kaltërtën:

/* Ngjyrat */ color: #ff0000; /* hex - e kuqe */ color: rgb(255, 0, 0); /* RGB */ color: hsl(0, 100%, 50%); /* HSL - me e lehte per te kuptuar */ color: red; /* emri i ngjyres */ /* Njësite e madhesise */ font-size: 16px; /* pixel - absolute */ font-size: 1rem; /* relative ndaj root (zakonisht 16px) */ font-size: 1.2em; /* relative ndaj prindit */ width: 50%; /* perqindje e prindit */ width: 50vw; /* 50% e gjeresise se ekranit */ /* Fontet nga Google Fonts */ @import url('https://fonts.googleapis.com/css2?family=Roboto'); body { font-family: 'Roboto', sans-serif; }
💡 Prefero rem: Nëse vendos madhësinë e tekstit si rem (jo px), faqja respekton preferencat e aksesueshmërisë të përdoruesit. Kjo është praktikë e mirë!
3

Flexbox — Paraqitja 1D

Mëso Flexbox dhe harro luftën me layout

2.5 orë

Flexbox e zgjidhi problemin më të vjetër të CSS — qendrimi i elementeve. Me 2 rreshta kodi, mund të qendrosh çdo gjë horizontalisht dhe vertikalisht:

/* Qendro cdo gje brenda .container */ .container { display: flex; justify-content: center; /* horizontal */ align-items: center; /* vertikal */ gap: 20px; /* hapesire mes elementeve */ } /* Navigim horizontal */ .navbar { display: flex; justify-content: space-between; /* logo majtas, menu djathtas */ align-items: center; padding: 15px 30px; } /* Karta te shperndarjes se barabarte */ .cards { display: flex; gap: 20px; flex-wrap: wrap; /* kalon ne rresht te ri nese nuk ka vend */ } .card { flex: 1; min-width: 200px; }
💡 Flexbox Froggy: Mëso Flexbox duke luajtur lojën falas "Flexbox Froggy" (flexboxfroggy.com). Shumë më mirë se të lexosh teori!
4

CSS Grid — Paraqitja 2D

Grids për faqe të plota dhe komplekse

2.5 orë

Nëse Flexbox është për rreshta dhe kolona (1D), Grid është për dy dimensione njëherësh. Perfekt për faqe të plota me header, sidebar, content, footer:

/* Grid 3 kolona te barabarta */ .grid { display: grid; grid-template-columns: 1fr 1fr 1fr; /* ose: repeat(3, 1fr) */ gap: 24px; } /* Layout i plote i faqes */ .faqja { display: grid; grid-template-areas: "header header" "sidebar content" "footer footer"; grid-template-columns: 250px 1fr; } header { grid-area: header; } .sidebar { grid-area: sidebar; } .content { grid-area: content; } footer { grid-area: footer; } /* Grid responsive pa media queries! */ .karta-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 20px; }
💡 auto-fit + minmax: Ky kombinim magjik krijon grids responsivë automatikisht — pa shkruar asnjë media query! Elementët rregullohen vetë sipas gjerësisë së ekranit.
5

Animacionet dhe Tranzicionet

Lëvizje dhe efekte me CSS

2 orë

Tranzicionet janë kalimi i butë nga një stil te tjetri — zakonisht kur hoverohet mbi element:

/* Tranzicion i thjeshte */ .butoni { background: blue; transition: all 0.3s ease; /* çfare, sa kohë, si */ } .butoni:hover { background: darkblue; transform: scale(1.05); } /* Animacion me @keyframes */ @keyframes pulso { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } } .zemra { animation: pulso 1s infinite; } /* Efekte te shpeshta */ .karte:hover { transform: translateY(-8px); } /* ngrihet */ .foto:hover { opacity: 0.8; } /* zbehet */
💡 Mos exagjeroj! Animacionet janë bukur kur janë të pakta dhe të qëlluara. Shumë lëvizje njëherësh e bëjnë faqen të duket amatore. Më pak = më mirë.
6

Design Responsiv

Faqe që duket mirë në çdo ekran

2 orë

Sot mbi 60% e trafikut vjen nga telefonat. Faqja jote DUHET të duket mirë në ekrane të vogla. Media queries janë zgjidhja:

/* Mobile-first: fillojmë me stilin e telefonit */ .grid { display: block; } /* kolona te veanta ne telefon */ /* Tablet: 600px e lart */ @media (min-width: 600px) { .grid { display: grid; grid-template-columns: 1fr 1fr; } } /* Desktop: 1024px e lart */ @media (min-width: 1024px) { .grid { grid-template-columns: repeat(3, 1fr); } .hero-title { font-size: 72px; } } /* Tipografi fluide - pa media queries! */ .titulli { font-size: clamp(24px, 5vw, 64px); /* min: 24px, prefered: 5% ekranit, max: 64px */ }
💡 Meta viewport! Mos harro të vendosësh <meta name="viewport" content="width=device-width, initial-scale=1.0"> në <head>. Pa këtë, telefoni tregon faqen të zvogëluar!
