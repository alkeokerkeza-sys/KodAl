Bazat e Python — Variablat dhe Tipet

Konfigurimi, variablat, numrat, stringjet, boolean

2 orë

Python është e ndryshme nga shumë gjuhë — nuk ka nevojë të deklarosh llojin e variablit, nuk ka ; në fund të rreshtave, dhe indentimi (hapësirat) janë pjesë e kodit. Kjo e bën Python shumë të lexueshme dhe të lehtë për fillestarët.

Për të filluar, instalo Python nga python.org dhe përdor VS Code si editor. Shkruaj skedarin e parë me .py dhe ekzekuto me python skedari.py në terminal.

# Python — variablat dhe tipet kryesore emri = "Arta" # string (tekst) mosha = 20 # int (numër i plotë) nota = 9.5 # float (numër dhjetor) aktiv = True # bool (e vërtetë / False) # Printimi i vlerave print("Pershendetje!") print(f"Emri im: {emri}, mosha: {mosha} vjec") # Input nga përdoruesi emri = input("Si te quajne? ") print(f"Mire se erdhe, {emri}!") # Operatorët matematikë print(10 + 3) # 13 print(10 / 3) # 3.333... print(10 // 3) # 3 (pjesëtim i plotë) print(10 % 3) # 1 (mbetja) print(2 ** 8) # 256 (fuqia)
💡 f-strings janë mënyra moderne! Shkruaj f"tekst {variabli}" — është shumë më e pastër se bashkimi me "tekst " + str(variabli). Gjithmonë përdor f-strings në kod modern!

Kontrolli i Rrjedhës — if, for, while

Kushtet, ciklet dhe logjika e programit

2 orë

Me kontrollin e rrjedhës i tregojmë programit "nëse kjo, bëj atë — ndryshe bëj tjetrin". Python e bën këtë me if/elif/else. Kujdes: indentimi 4 hapësira është i DETYRUESHËM — ndryshe programi jep gabim!

# if / elif / else nota = 8.5 if nota >= 9: print("Shkëlqyeshëm! ⭐") elif nota >= 6: print("Kaloi!") else: print("Duhet më shumë punë...") # for loop — për çdo element kurset = ["HTML", "CSS", "Python"] for kurs in kurset: print(f"Duke mësuar: {kurs}") # range() — gjeneron sekuencë numrash for i in range(5): # 0, 1, 2, 3, 4 print(i) for i in range(1, 11): # 1 deri 10 print(f"{i} x 5 = {i*5}") # while loop — derisa plotësohet kushti numri = 0 while numri < 5: print(f"Numri: {numri}") numri += 1 # += shton 1 çdo herë # List comprehension — Python-ike dhe elegante! katratet = [x**2 for x in range(1, 6)] print(katratet) # [1, 4, 9, 16, 25] # Fjalë kyçe break dhe continue for i in range(10): if i == 3: continue # kalo 3-shin if i == 7: break # ndalon te 7 print(i) # 0,1,2,4,5,6
💡 range(start, stop, step): range(10, 0, -1) numëron nga 10 deri 1 mbrapsht. range(0, 20, 2) jep çiftet 0,2,4,...18. Shumë i dobishëm!
3

Funksionet — Kodi i Ripërdorshëm

def, parametrat, return, scope dhe lambda

2.5 orë

Funksionet janë blloqe kodi me emër — i shkruajmë NJË herë dhe i përdorim sa herë të duhet. Parimi kryesor është DRY — Don't Repeat Yourself. Nëse shkruajmë të njëjtin kod dy herë, e bëjmë funksion.

# Funksion i thjeshtë def pershendet(emri): print(f"Pershendetje, {emri}!") pershendet("Arta") # → Pershendetje, Arta! pershendet("Blerim") # → Pershendetje, Blerim! # Funksion me return def shto(a, b): return a + b rezultati = shto(5, 7) print(rezultati) # 12 # Vlera default — nëse nuk jepet argument def pershkruaj(emri, mosha=18): return f"{emri} eshte {mosha} vjec" print(pershkruaj("Arta")) # mosha = 18 (default) print(pershkruaj("Blerim", 22)) # mosha = 22 # *args — numër i panjohur argumentesh def shto_te_gjitha(*numrat): return sum(numrat) print(shto_te_gjitha(1, 2, 3, 4, 5)) # 15 # Lambda — funksion i shkurtër njëherësh dyfish = lambda x: x * 2 print(dyfish(7)) # 14 katrore = lambda x, y: x * y print(katrore(4, 5)) # 20
💡 Scope: Variablat brenda funksionit janë "lokale" — nuk ekzistojnë jashtë tij. Për të modifikuar variabla globale brenda funksionit, përdor fjalën kyçe global, por shmangje kur të mundesh — e bën kodin të vështirë për debug.

Strukturat e të Dhënave

Lista, tuple, fjalor (dict), bashkësi (set)

2.5 orë

Python ka katër struktura kryesore të të dhënave. Të diturit çka të përdorësh e bën diferencën mes kodit të mirë dhe atij kaotik.

# LIST — koleksion i renditur dhe i ndryshueshem kurset = ["HTML", "CSS", "Python"] kurset.append("JavaScript") # shto në fund kurset.insert(0, "Git") # shto në pozicion kurset.remove("CSS") # fshij sipas vlerës print(kurset[0]) # elementi i parë print(kurset[-1]) # i fundit print(kurset[1:3]) # slice: nga 1 deri 3 print(len(kurset)) # numri i elementeve # DICTIONARY — çelës:vlerë, si JSON studenti = { "emri": "Arta", "mosha": 20, "nota": 9.5, "kurset": ["Python", "SQL"] } print(studenti["emri"]) # Arta studenti["qyteti"] = "Tiranë" # shto çelës të ri studenti.get("mbiemri", "N/A") # nëse nuk ekziston → N/A # Iterim mbi dict for celesi, vlera in studenti.items(): print(f"{celesi}: {vlera}") # TUPLE — si lista por i pandryshueshëm koordinatat = (41.33, 19.82) # Tiranë lat/lon print(koordinatat[0]) # 41.33 # SET — elemente unike, pa rend gjuhe = {"Python", "Python", "JS", "SQL"} print(gjuhe) # {"Python", "JS", "SQL"} — duplikati hiqet!
💡 Kur cfarë të përdorësh? LIST kur renditja ka rëndësi dhe ndryshon. DICT kur ke çelës-vlerë (si objekt JSON). TUPLE kur të dhënat nuk ndryshojnë kurrë (koordinata, data). SET kur duhet të kontrollosh nëse elementi ekziston — SET është jashtëzakonisht i shpejtë për këtë!

Skedarët dhe Trajtimi i Gabimeve

Lexo/shkruaj skedarë, try/except, JSON

2 orë

Programet reale lexojnë dhe shkruajnë skedarë — CSV, TXT, JSON. Python e bën këtë shumë të lehtë me open(). Gjithmonë përdor with — e mbyll skedarin automatikisht.

# Shkruaj në skedar with open("skedari.txt", "w", encoding="utf-8") as f: f.write("Rreshti i parë\n") f.write("Rreshti i dytë\n") # Lexo të gjithë skedarin with open("skedari.txt", "r", encoding="utf-8") as f: permbajtja = f.read() print(permbajtja) # Lexo rresht pas rreshti with open("skedari.txt") as f: for rresht in f: print(rresht.strip()) # heq \n # JSON — shkruaj dhe lexo import json te_dhenat = {"emri": "Arta", "nota": 9.5} with open("data.json", "w") as f: json.dump(te_dhenat, f, indent=2) with open("data.json") as f: te_ngarkuara = json.load(f) print(te_ngarkuara["emri"]) # Arta # try/except — trajtimi i gabimeve try: numri = int(input("Shkruaj nje numer: ")) print(100 / numri) except ValueError: print("Kjo nuk ishte numër i vlefshëm!") except ZeroDivisionError: print("Pjesëtimi me zero nuk lejohet!") finally: print("Ky blok ekzekutohet gjithmonë.")
💡 Modalitetet e hapjes: "r" = lexo (default), "w" = shkruaj (fshi të vjetrat), "a" = shto në fund, "r+" = lexo dhe shkruaj. Gjithmonë jep encoding="utf-8" për të shmangur probleme me shkronjat shqipe!

OOP — Programimi i Orientuar nga Objekti

Klasat, objektet, trashëgimia, encapsulimi

3 orë

OOP është mënyra e organizimit të kodit rreth "objekteve" — strukturave që bashkojnë të dhëna dhe funksione në një vend. Kjo e bën kodin shumë më të organizuar dhe të ripërdorshëm.

# Klasa — modeli/blloku class Studenti: # __init__ thirret automatikisht kur krijohet objekti def __init__(self, emri, mosha, nota): self.emri = emri # atribut self.mosha = mosha self.nota = nota def prezanto(self): # metodë print(f"{self.emri}, {self.mosha} vjec, nota: {self.nota}") def ka_kaluar(self): return self.nota >= 6.0 # Krijo objekte (instance) s1 = Studenti("Arta", 20, 9.5) s2 = Studenti("Blerim", 22, 5.0) s1.prezanto() # Arta, 20 vjec, nota: 9.5 print(s2.ha_kaluar()) # False # Trashëgimia — klasa fëmijë merr gjithçka nga prindi class StudentDoktorature(Studenti): def __init__(self, emri, mosha, nota, tema): super().__init__(emri, mosha, nota) # thirr prindi self.tema = tema def prezanto(self): # override metodën super().prezanto() print(f"Tema: {self.tema}") doktor = StudentDoktorature("Çlirim", 28, 10, "AI në Arsim") doktor.prezanto()
💡 self: Çdo metodë e klasës ka self si parametrin e parë — është referenca ndaj objektit aktual. Python e kalon automatikisht, nuk duhet ta japësh ti kur e thërret metodën.

🎯 Projekti Final — Skript Real

Ndërto diçka të dobishme duke i bashkuar të gjitha

6 orë

Zgjidh NJË nga projektet — ai që ta zë zemra. Të gjithë janë real dhe mund të vendosen në GitHub!

🗒️ Menaxhues Detyrave (Todo App): Lista todo me komanda shto/fshi/shënoji si të bëra. Ruan të dhënat në skedar JSON.

📊 Analizues CSV: Merr një skedar CSV (çmimet e produkteve, notat e studentëve) dhe llogarit mesatare, min, max, tregon statistika.

🎲 Lojë Numrash: Kompjuteri zgjedh numër 1-100, ti gjen me hints "shumë lartë/ulët". Ruan rezultatin më të mirë.
